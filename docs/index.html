Bullframe CSS v6.0.0

Complete documentation for Bullframe CSS framework

Getting Started

Bullframe CSS v6.0.0 is a lightweight CSS framework built with native CSS and PostCSS. No Sass required!

CDN Installation

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bullframe.css@6/dist/css/bullframe.min.css">

npm Installation

npm install bullframe.css

CSS Variables

All variables use the --bf-* prefix and are defined in :root.

Color Variables

:root { --bf-white: rgb(255 255 255); --bf-dark: rgb(34 34 34); --bf-blue: rgb(0 123 255); /* ... and more */ }

Spacing Variables

:root { --bf-spacing-sm: 0.25rem; --bf-spacing-md: 0.5rem; --bf-spacing-lg: 1rem; /* ... */ }
