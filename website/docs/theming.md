# Theming

Bullframe CSS provides multiple theme variants and easy customization options.

## Theme Variants

### Light Theme (Default)

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bullframe.css@6/dist/css/bullframe.min.css">
```

### Dark Theme

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bullframe.css@6/dist/css/bullframe-dark.min.css">
```

### System Default Theme

Automatically switches between light and dark based on user's system preference:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bullframe.css@6/dist/css/bullframe-system-default.min.css">
```

## Classless Variants

All themes are also available in classless versions (no utility classes):

- `bullframe-classless.css` - Light theme, classless
- `bullframe-classless-dark.css` - Dark theme, classless
- `bullframe-classless-system-default.css` - System default, classless

## Customization

See the [Customization](/docs/theming/customization) guide for details on customizing themes.
